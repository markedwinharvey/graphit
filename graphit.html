graphit is a visualization tool for simple mathematical functions.
Power functions (pow) and basic trigonometric functions (sin, cos, tan) are supported.
pow(x,2) is equivalent to x2. Trig functions are written as sin(x), cos(3*n), tan(x/n).
Graphit will draw mathematical curves (as a series of points) for a given function of x.
Curves can be plotted against a second variable n to generate some interesting behavior.
The function, limits and increments of x and n, and point size can all be modified.
Group statements by parentheses and use * for multiplication.
An example formula has been preloaded into the formula field. Click graphit to see how it works.
The graphing function can slow as the graph becomes more complicated.
Smaller increments result in more intricate/full designs, but also require more time to complete.
Press stop to stop processing and clear to clear graphing window.
Graphs can be overlaid, but processing speed will suffer considerably (use clear!).
